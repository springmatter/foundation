## `<button>` only Buttons

If you just need to make something clickable and focusable, a standard `<button>` tag is useful. `<button>` should be used instead of `<a>` or `<router-link>` when the clickable element doesn't toggle a route change or a page location change on a page of text. Using `<button>` to wrap clickable elements makes them more accessible by default.
