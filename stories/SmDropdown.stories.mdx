# SmDropdown

A simple component for selecting an option from a dropdown list.

<Props of={SmDropdown} />

## Simple Dropdown

Dropdowns with more than 5 options are scrollable.

<Preview>
  <Story name="simple" height="300px">
    {{
      components: { SmDropdown },
      template: `
      <div class="flex">
        <SmDropdown class="m-2" style="width:300px;" 
            :options="['apple', 'pear', 'banana', 'mango', 'guava', 'lemon', 'grapefruit']"/>
        <SmDropdown class="m-2" style="width:300px;" 
            displayKey="name"
            :options="[{ name: 'Sally', type: 'dog', color: 'white'},
                       { name: 'Wanda', type: 'dog', color: 'brown'},
                       { name: 'Jack', type: 'cat', color: 'black'}, 
                       { name: 'Pepper', type: 'dog', color: 'mixed'}, 
                       { name: 'Lucy', type: 'cat', color: 'white'}]" />
      </div>
      `
    }}
  </Story>
</Preview>

## Searchable Dropdown

For an array of Objects, the search is performed over all keys.
